A null array is initialized with fixed rows-columns size corresponding to average hourly UVI values, 24 hours per day, 365 days a year, for every year in the past two decades. Thereby, daily \(\overline{UVI}\) values were hourly compared with \(i\)from 0h to 23h for each date and station. Each position keeps a value into array and it was permuted by maximum value swiping all measurements over MCMA along years:
\[\overline{UVI}_{i,d,y}^s\ \le\overline{UVI}_{i,d,y}^{s'}\ \to\overline{UVI}_{i,d}^{\max}\text{ in the period 2000-2019}\]
simultaneously applying to \(i\), \(d\) and \(y\) the same values in both sides of inequality. The \(s\) and \(s'\)represent the data set of the differents stations. This process is repeated until building a matrix  \(\overline{UVI}_{i,d}^{\max}\)with maximum UV Index values per hour in all the period 2000-2019.
