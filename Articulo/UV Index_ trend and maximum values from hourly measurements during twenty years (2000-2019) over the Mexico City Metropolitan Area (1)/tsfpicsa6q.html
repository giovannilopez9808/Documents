The algorithm create a matrix with the maximums values of all stations, all years sorted by date and hour. The empty spaces are ignored in the calculation as well as invalid values product of maintenance of the equipments. Thereby,  \(\overline{UVI}\) maximum values of each position keeps a value into array and it was permuted by maximum value swiping all measurements over MCMA along years:
\[\overline{UVI}_{i,d,m,y}^s\ \le\overline{UVI}_{i,d,m, y}^{s'}\ \to\overline{UVI}_{i,d}^{\max}\text{ 2000-2019}\]
simultaneously applying to \(i\), \(d\), \(m\) and \(y\) the same values in both sides of inequality. The \(s\) and \(s'\)represent the data set of the differents stations. This process is repeated until building a matrix  \(\overline{UVI}_{i,d}^{\max}\)with maximum UV Index values per hour each date as representative in all the period 2000-2019.
