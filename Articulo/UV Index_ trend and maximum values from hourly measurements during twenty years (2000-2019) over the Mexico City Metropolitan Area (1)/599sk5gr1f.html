At least in 5 stations, this daily systematic operation was carried out during two decades. The averages hourly UV Index (\(\overline{UVI}\)) conform the 2,000,000 values synthesized from historical database.

Measurements analysis

Main analysis was performed to identify the maximum UVI values measured  over MCMA in the complete 2000-2019 period. We developed a Python script to process the data set. The algorithm collects the information by station (\(s\)) and create an individial list with the elements: date day-month (\(d\)), year (\(y\)), hour (\(i\)) and \(\overline{UVI}\). All the empty spaces are ignored in the calculation as well as invalid values product of maintenance of the equipments. A null array is initialized with fixed rows-columns size corresponding to average hourly UVI values, 24 hours per day, 365 days a year, for every year in the past two decades. Thereby, daily \(\overline{UVI}\) values were hourly compared with \(i\)from 0h to 23h for each date and station. Each position keeps a value into array and it was permuted by maximum value swiping all measurements over MCMA along years:
\[\overline{UVI}_{i,d,y}^s\ \le\overline{UVI}_{i,d,y}^{s'}\ \to\overline{UVI}_{i,d}^{\max}\text{ in the period 2000-2019}\]
simultaneously applying to \(i\), \(d\) and \(y\) the same values in both sides of inequality. The \(s\) and \(s'\)represent the data set of the differents stations. This process is repeated until building a matrix  \(\overline{UVI}_{i,d}^{\max}\)with maximum UV Index values per hour in all the period 2000-2019.
